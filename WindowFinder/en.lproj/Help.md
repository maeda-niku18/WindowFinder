# How to Use Window Finder

Window Finder helps you quickly find and bring open windows to the front.

## Basic Controls

- Press **⌃⌥Space** to show the finder. You can change the shortcut in Settings.
- Press the shortcut again while the finder is open to refresh the window list.
- Use **↑ ↓ ← →** to select a window, then press **Enter** to bring it to the front.
- Press **Esc** to close the finder.
- You can also click a card to activate that window.

## Search

- Type in the search field to filter by app name or window title.
- The first result is selected automatically, so you can type and press Enter to switch quickly.

## Thumbnails

- Screen Recording permission is required to show window thumbnails.
- Minimized windows cannot provide a live image. Window Finder shows the last captured image when available, or falls back to the app icon.

## Permissions

- **Accessibility**: Used to read the window list and bring windows to the front.
- **Screen Recording**: Used to show thumbnails.
