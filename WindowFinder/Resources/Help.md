# Window Finder の使い方

開いているウィンドウを最速で探して呼び出すためのユーティリティです。

## 基本操作

- **⌃⌥Space**（既定）でランチャーを表示します。ショートカットは設定で変更できます。
- 表示中にもう一度ショートカットを押すと、**最新のウィンドウ情報に取り直し**ます。
- **↑ ↓ ← →** でウィンドウを選択し、**Enter** で前面に呼び出します。
- **Esc** で閉じます。
- マウスでカードをクリックしても呼び出せます。

## 検索

- 検索欄に入力すると、**アプリ名・ウィンドウタイトル**で絞り込みます（部分一致）。
- 先頭の候補が自動選択されるので、入力 → Enter で素早く移動できます。

## サムネイル

- ウィンドウのサムネイルには **画面収録権限** が必要です。
- **最小化中のウィンドウ**はライブでは取得できないため、最後に表示したときの画像（あれば）か、アプリアイコンを表示します。

## 権限について

- **アクセシビリティ権限**（必須）: ウィンドウの一覧取得・前面化に使用します。
- **画面収録権限**（任意）: サムネイル表示に使用します。

## 設定

- **表示順**: ウィンドウ数順 / アプリ名順
- **サムネイルの大きさ** / **列数** / **ウィンドウの大きさ**
- **ショートカットの変更**
