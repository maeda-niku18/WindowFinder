# Window Finder

開いているウィンドウを一覧から選んで、すばやく前面に呼び出すための macOS 用ユーティリティです。

たくさんのアプリやウィンドウを開いていると、「さっきのウィンドウがどこに行ったか分からない」「別の Space にあって探しにくい」という状況になりがちです。Window Finder は、画面中央にウィンドウ一覧を表示し、探しているウィンドウをその場で選べるようにします。

![Window Finderで開いているウィンドウを一覧から選ぶ画面](docs/images/window-finder-overview.png)

サムネイル付きの一覧から、開いているウィンドウを見ながら選べます。キーボードでもマウスでも操作でき、選んだウィンドウは最小化されていても復元して前面に呼び出します。

## できること

- 開いているウィンドウをサムネイル付きで一覧表示
- アプリ名やウィンドウタイトルで検索
- 矢印キーで選択し、Enter で呼び出し
- マウスクリックでウィンドウを呼び出し
- 最小化中のウィンドウを復元して前面に表示
- グローバルショートカットでいつでも表示
- 表示順、サムネイルサイズ、列数、高さ、自動スクロールを設定

## 使い方

1. **⌃⌥Space** を押して Window Finder を表示します。
2. 一覧から呼び出したいウィンドウを選びます。
3. **Enter** またはクリックで、そのウィンドウを前面に呼び出します。

表示中にもう一度 **⌃⌥Space** を押すと、ウィンドウ一覧を更新します。閉じるときは **Esc** を押します。

## ショートカット

| 操作 | 内容 |
| --- | --- |
| **⌃⌥Space** | Window Finder を表示、表示中は一覧を更新 |
| **↑ ↓ ← →** | ウィンドウを選択 |
| **Enter** | 選択中のウィンドウを呼び出す |
| **Esc** | 閉じる |

既定のショートカットは **⌃⌥Space** です。設定画面から別のキー組み合わせに変更できます。

## 検索

検索欄に入力すると、アプリ名またはウィンドウタイトルで絞り込めます。候補の先頭が自動で選択されるため、文字を入力して Enter を押すだけで目的のウィンドウに移動できます。

## 権限

- **アクセシビリティ権限**: ウィンドウ一覧の取得と、選択したウィンドウの前面表示に使用します。必須です。
- **画面収録権限**: サムネイル表示に使用します。未許可の場合はアプリアイコンで代替します。

## 動作環境

- macOS 14.0 以上

対応バージョンは `project.yml` の `deploymentTarget.macOS` で確認済みです。

## インストール

1. [リリースページ](https://github.com/maeda-niku18/WindowFinder/releases) から最新の DMG をダウンロードします。
2. DMG を開き、`WindowFinder.app` を「アプリケーション」フォルダへドラッグします。
3. 初回起動時に、アクセシビリティ権限を許可します。
4. サムネイルを表示したい場合は、画面収録権限も許可します。

公証済みの DMG として配布しています。新しいバージョンがある場合は、アプリ内で更新を通知します。

## 開発

プロジェクトは [XcodeGen](https://github.com/yonaskolb/XcodeGen) で生成します。設定は `project.yml` を編集してください。

```bash
xcodegen generate
open WindowFinder.xcodeproj
```

依存ライブラリ:

- [KeyboardShortcuts](https://github.com/sindresorhus/KeyboardShortcuts): グローバルショートカット
- [Sparkle](https://github.com/sparkle-project/Sparkle): 自動アップデート

## 配布

```bash
# project.yml の MARKETING_VERSION を更新してから
./release.sh 1.1.0
```

署名、DMG 作成、公証、ステープル、appcast 更新、GitHub Releases へのアップロードまでを自動で行います。
